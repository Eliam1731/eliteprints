# ElitePrints

## Tools (could change)
- __Frontend:__ Astro w react/vue to islands  
- __Backend:__ Bun w ElysiaJS  
- __DB:__ Supabase (postgres) to manage dashboard auth and status of subscriptions  
- __Lemon Squeezy:__ MoR to manage payments and subscriptions related to money  
- __Hosting:__ Could be a VPS with DigitalOcean or PaaS like Render, GithubPages, Vercel - needs to be clarified  

## Considerations
This will be a monorepo, which means, running any command in `/package.json` will run nested `package.json` commands from apps/*
